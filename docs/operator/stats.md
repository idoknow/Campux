# 统计看板

统计页帮助运营者判断当前校园墙的投稿、审核、发布和机器人状态。

![统计看板](/screenshots/stats-dashboard.png)

## 时间范围

看板支持切换：

- 近 7 天
- 近 14 天
- 近 30 天
- 近 90 天

曲线图可以点击某一天查看具体数值。较长时间范围会保留页面宽度，不会让整个内容区横向滚动。

## 核心指标

| 指标 | 用途 |
| --- | --- |
| 总稿件 | 判断墙的总体活跃度 |
| 投稿用户 | 判断有多少不同账号参与投稿 |
| 平均审核 | 判断审核是否积压 |
| 发布成功率 | 判断 QZone 发布链路是否稳定 |
| 匿名比例 | 判断投稿氛围 |
| 配图比例 | 判断内容类型 |
| 封禁状态 | 判断治理压力 |

## 发布质量

发布质量区域会按发布目标展示成功率、失败数和风控间隔。失败率高时优先检查：

1. cookies 是否可用。
2. 是否触发 QZone 风控。
3. 图片上传是否被 QZone 拒绝。
4. 发布目标是否错误配置为必需。

## 操作审计

操作审计显示最近管理动作，例如更新发布目标、刷新 cookies、修改成员身份、封禁用户等。它适合用于追踪“是谁改了什么配置”。
